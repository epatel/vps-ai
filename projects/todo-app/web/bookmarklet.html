Add to Todos — Bookmarklet

A one-click bookmark that turns the page you're viewing into a new todo.

Drag this button to your bookmarks bar:

+ Add to Todos

Drag the button to your bookmarks bar. Clicking it from this page will just create a todo for this page itself — handy as a smoke test.

How to use

  1. Drag the button above to your browser's bookmarks bar.
  2. Visit any web page you want to remember.
  3. Click the bookmark — Todos opens in a new tab with the title and URL pre-filled.
  4. Confirm in the dialog to save.

Browser-specific setup

Chrome / Edge / Firefox / Safari (desktop): drag-and-drop works as described.

iOS Safari: drag-and-drop isn't available. Instead:

  1. Bookmark any page (tap Share → Add Bookmark).
  2. Open Bookmarks, tap Edit, select the new bookmark.
  3. Replace its URL with the code below, rename it to e.g. todo, save.
  4. To use: type todo in the address bar and tap the bookmark suggestion.

Android Chrome: bookmarklets are unreliable — install this app as a PWA instead (menu → "Add to Home Screen"). Then "Todos" appears in the system share sheet from any app.

Manual code

If drag-and-drop doesn't work, copy this and paste it as a bookmark URL:

javascript:(function(){var u=encodeURIComponent(location.href);var t=encodeURIComponent(document.title||'');var s='';try{s=encodeURIComponent(String(window.getSelection()||''));}catch(e){}var q='title='+t+'&url='+u+(s?'&text='+s:'');window.open('https://ai.memention.net/todo-app/?'+q,'_blank');})();

What it does

The bookmarklet reads three things from the current page and opens the Todos app with them as query parameters:

  • title — the page's <title>
  • url — the page's address
  • text — any text you have selected (optional)

Todos picks these up via the same Web Share Target handler the PWA uses, opens the Add Todo dialog pre-filled, and cleans the query string from the URL.
